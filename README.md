# WaikTech React Counter App

This is a simple **counter app** built with **React JS, Redux Toolkit, and Tailwind CSS**.

## Features
- Increment / Decrement counter
- Prevent negative values
- Professional, modern UI
- Redux global state management
- Futuristic design with hover/active effects

## Tech Stack
- React JS + TypeScript
- Redux Toolkit
- Tailwind CSS
